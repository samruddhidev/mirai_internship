# MirAi School of Technology Internship Projects

This repository contains the projects I developed during my internship at MirAi School of Technology. These projects were built using Python and Streamlit to understand the basics of interactive web application development.


# Project 1 – Smart Calculator

## Overview

The Smart Calculator is a simple Streamlit application that performs basic arithmetic operations. It allows users to enter two numbers, choose an operation, and instantly view the result.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Division by zero validation
- Simple and interactive user interface

## Concepts Learned

- Using Streamlit to create web applications
- `st.title()`
- `st.number_input()`
- `st.radio()`
- `st.button()`
- Conditional statements (`if`, `elif`, `else`)
- Displaying results using `st.success()`


# Project 2 – Feedback Portal

## Overview

The Feedback Portal is a Streamlit-based form that collects a user's name, email address, and feedback. After submission, it validates the input and displays useful statistics about the message.

## Features

- Name input
- Email input
- Feedback input
- Input validation
- Success message
- Estimated token count
- Word count
- Character count
- Feedback summary

## Concepts Learned

- `st.text_input()`
- `st.text_area()`
- Form validation
- String manipulation
- Token estimation
- Displaying information using `st.success()`, `st.info()`, and `st.error()`

# How to Run

### 1. Install Streamlit

```bash
pip install streamlit
```

### 2. Navigate to the project folder

```bash
cd path/to/project
```

### 3. Run Project 1 (Calculator)

```bash
streamlit run calculator.py
```

### 4. Run Project 2 (Feedback Portal)

```bash
streamlit run app.py
```
